`type: "title"`

`text: текст` - Текст заголовка;

`textSize?: 's' | 'm' | 'l'` - Размер шрифта заголовка;

`url?: string` - урл, по которому можно перейти при клике на заголовок, автоматически в конце добавляется стрелочка

**Отступы:**

`textSize s - top: m`

`textSize m - top: l`

`textSize l - top: xl`
