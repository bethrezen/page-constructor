Блок "Табы"

`type: tabs-block` - заголовок;

`title: Title` - заголовок;

    `items: TabsBlockItem[]` - описание таба

      > `tabName: string` - заголовок таба;

      > `image: string` - изображение;

      > `caption?: string`  - подпись по картинкой;

      > `text: string` - текст;

      > `link?: Link`  - ссылка под текстом;
